# ✨ To-Do Manager

직관적이고 아름다운 할일 관리 웹 애플리케이션

## 🚀 주요 기능

### 📋 할일 관리
- **할일 추가/수정/삭제**: 제목, 메모, 날짜, 시간을 포함한 상세한 할일 관리
- **완료 체크**: 체크박스로 간편하게 완료 상태 전환
- **우선순위 정렬**: 미완료 할일 우선, 날짜 및 시간 순으로 자동 정렬
- **기한 알림**: 기한이 지난 할일은 빨간색으로 강조

### 📁 폴더 시스템
- **커스텀 폴더**: 업무, 개인, 공부 등 원하는 카테고리 생성
- **자동 색상 할당**: 각 폴더마다 파스텔 톤 색상 자동 지정
- **폴더 필터링**: 특정 폴더의 할일만 선택적으로 보기

### 📅 스마트 필터링
- **오늘**: 오늘 할 일만 표시
- **이번주**: 이번 주에 할 일 표시
- **나중에**: 이번 주 이후의 할일 표시
- **전체**: 모든 할일 표시

### 🌅 대시보드
- 상단에 오늘의 일정을 카드 형태로 한눈에 표시
- 시간 순으로 정렬된 오늘의 할일
- 클릭하여 바로 수정 가능

### 💾 데이터 영속성
- 로컬 스토리지를 활용한 자동 저장
- 브라우저를 닫아도 데이터 유지
- 새로고침 후에도 모든 정보 보존

## 🎨 디자인

- **심플하고 모던한 UI**: 깔끔한 인터페이스
- **파스텔 톤 컬러**: 눈에 편안한 색상 팔레트
- **부드러운 애니메이션**: Hover 효과 및 전환 애니메이션
- **반응형 디자인**: 데스크톱, 태블릿, 모바일 완벽 지원

## 🛠️ 기술 스택

- **React**: UI 라이브러리
- **Vite**: 빠른 개발 환경
- **CSS Modules**: 스코프가 있는 스타일링
- **JavaScript ES6+**: 최신 JavaScript 문법

## 📦 설치 및 실행

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

개발 서버가 시작되면 브라우저에서 `http://localhost:5173` (또는 표시된 포트)로 접속하세요.

### 프로덕션 빌드
```bash
npm run build
```

### 프로덕션 미리보기
```bash
npm run preview
```

## 📱 사용 방법

1. **할일 추가**: 우측 상단의 "+ 할 일 추가" 버튼 클릭
2. **폴더 관리**: 좌측 사이드바에서 폴더 추가/삭제
3. **필터링**: 상단 탭으로 오늘/이번주/나중에 할일 필터링
4. **할일 수정**: 할일 카드 클릭하여 수정
5. **완료 체크**: 체크박스 클릭으로 완료 표시
6. **삭제**: 휴지통 아이콘 클릭

## 📂 프로젝트 구조

```
src/
├── components/          # React 컴포넌트
│   ├── Dashboard.jsx   # 오늘의 일정 대시보드
│   ├── TabFilter.jsx   # 카테고리 탭 필터
│   ├── TodoList.jsx    # 할일 목록
│   ├── TodoItem.jsx    # 개별 할일 항목
│   ├── TodoForm.jsx    # 할일 추가/수정 폼
│   └── FolderManager.jsx # 폴더 관리
├── hooks/
│   └── useLocalStorage.js # 로컬 스토리지 훅
├── utils/
│   └── dateUtils.js    # 날짜 유틸리티 함수
├── App.jsx             # 메인 앱 컴포넌트
├── App.module.css      # 앱 스타일
└── index.css           # 전역 스타일
```
